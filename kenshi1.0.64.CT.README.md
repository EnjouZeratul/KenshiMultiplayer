# Kenshi 1.0.64 CT 表说明

本 CT 表与 KenshiMultiplayer 共用偏移。以下说明**已修复项具体服务于哪些功能**，以及**哪些功能可用、哪些可能有问题**。

---

## 一、脚本说明

### recordStatus（按 C 记录）— 保持原样 `[rcx+CC]`

| 项目 | 说明 |
|------|------|
| **服务于** | 「通过按 C 开属性页面记录角色属性基址」— 为整个「男儿当自强」模块提供 A00/A08/A10 基址 |
| **状态** | 原版正确，未修改 |

### lockHP（鼠标左键选取）— 已修复

| 项目 | 说明 |
|------|------|
| **服务于** | 「通过鼠标左键选取角色记录角色属性基址」— 点击角色即可记录 A00/A08/A10 |
| **修复前问题** | 原指令 `movss xmm0,[rsi+70]` 表明 rsi 为角色指针，脚本误用 rcx 保存，导致选中角色时 A00 错误 |
| **修复后** | 改为使用 rsi 捕获角色基址，鼠标点击选中角色可正确读取角色信息 |

---

## 二、功能状态概览

### 可用（依赖 recordStatus，修复后应正常）

| 功能 | 说明 |
|------|------|
| 角色姓名 | A00 +18 → +10 |
| 钱 | A00 +10 → +10 → +80 → +88 |
| 属性、技能 | A00 + 各偏移（80, 84, 88...） |
| 血量、部位血量 | A00 +70 或 A00 +08 +1a0 链 |
| 饥饿、昏迷、犯罪时间 | 各自偏移链 |
| 金钱设为 10 万 | 依赖金钱链 |
| 明帝内裤（体型） | 独立 RVA 4175E8，不依赖 A00 |

### 可能有问题（需单独验证）

| 功能 | 可能问题 |
| **建造无需材料** | CT 用 43090A，game_types 用 54A41A，RVA 不一致 |
| **快速建造** | CT 用 43FC7A，game_types 用 55978A，RVA 不一致 |
| **1.0.68 用户** | 本 CT 为 1.0.64，1.0.68 需单独 CT 或改回 0xCC 等偏移 |

### 未改动（保持原样）

以下脚本**未被修改**，保持原有逻辑：

- lockHP 已修复（见上文），其余：
- 圣火令、乾坤大挪移、久阳初现、久阳在天
- 经验倍数、建造、肢解、悬赏、好感
- 装备修改、物品相关

## 三、与 KenshiMultiplayer 共用

- 角色基址：A00 = Character
- 金钱链：A00 +0x10 → +0x10 → +0x80 → +0x88
- 姓名：A00 +0x18 → +0x10
- 位置：A00 +0x20..+0x28
- HP：A00 +0x70
